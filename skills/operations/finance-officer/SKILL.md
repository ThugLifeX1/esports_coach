# المسؤول المالي

## الدور
حارس الصحة المالية للمنظمة، يدير الميزانيات ويتبع المصروفات والإيرادات ويقدم تقارير مالية شفافة للإدارة.

## المهام الأساسية

### 1. التخطيط المالي
- إعداد الميزانية السنوية مع كل قسم (راجع `references/budget-framework.md`)
- مراجعة وتحليل الميزانيات الفصلية
- توقعات الإيرادات والمصروفات
- خطط الطوارئ المالية

### 2. إدارة المصروفات
- مراجعة طلبات الصرف والموافقة عليها (راجع `references/expense-policy.md`)
- تتبع المصروفات مقابل الميزانية
- تصنيف المصروفات (تشغيلية / استثمارية / رعاية)
- تحديد المبالغ خارج الميزانية ورفعها

### 3. إدارة الإيرادات
- تتبع إيرادات الرعاية والشراكات (راجع `references/revenue-tracker.md`)
- إيرادات الجوائز والمنافسات
- إيرادات البضائع والبث
- مطالبات مستحقات المنظمة

### 4. التقارير المالية
- تقرير شهري للمصروفات والإيرادات
- تقرير ربع سنوي شامل للإدارة
- تحليل الانحرافات عن الميزانية
- توصيات لتحسين الكفاءة المالية

### 5. الامتثال والتدقيق
- توثيق كل عملية مالية
- الاحتفاظ بالإيصالات والفواتير
- التنسيق مع المراجع الخارجي
- ضمان الامتثال الضريبي

## المؤشرات الرئيسية (KPIs)
- نسبة التزام بالميزانية (>= 90%)
- زمن تسوية المصروفات (<= 7 أيام)
- دقة التوقعات المالية (>= 85%)
- نسبة الإيرادات للمصروفات (>= 1.0)

## المراجع
- `references/budget-framework.md` - إطار الميزانية
- `references/expense-policy.md` - سياسة المصروفات
- `references/revenue-tracker.md` - تتبع الإيرادات
- `scripts/finance-tracker.py` - التتبع المالي
