# سياسة المصروفات

## تصنيف المصروفات

### مصروفات تشغيلية (OPEX)
- رواتب ومستحقات
- إيجار ومرافق
- إنترنت واتصالات
- تموين ونظافة
- اشتراكات برمجية
- سفر ولوجستيات

### مصروفات استثمارية (CAPEX)
- أجهزة ومعدات جديدة
- تحسين المرافق
- تطوير البرمجيات
- شراء أصول

## إجراءات طلب الصرف

### طلب مصروف عادي
1. الموظف يعبئ نموذج طلب الصرف
2. مدير القسم يعتمد
3. المسؤول المالي يراجع ويوافق
4. التحويل أو الدفع النقدي
5. تسليم الإيصال خلال 7 أيام

### طلب مصروف طارئ
1. اتصال هاتفي مع المسؤول المالي
2. موافقة شفهية + توثيق لاحق
3. تنفيذ الصرف خلال 24 ساعة
4. استكمال الإجراءات خلال 3 أيام

## قواعد الإيصالات
- إيصال أصلي أو إلكتروني معتمد
- اسم المورد مطابق للفاتورة
- تاريخ الإيصال خلال 30 يوم
- لا سداد نقدي فوق 5,000 (تحويل فقط)
