# قائمة الحجوزات

## قائمة التحقق لكل رحلة

### التذاكر
- [ ] حجز تذاكر الطيران/القطار لكل مشارك
- [ ] تأكيد أوزان الأمتعة المسموحة
- [ ] إضافة أمتعة إضافية للأجهزة (شاشات، أجهزة)
- [ ] اختيار المقاعد (ممر للراحة)
- [ ] طباعة/تحميل boarding passes

### الإقامة
- [ ] حجز غرف بنفس الفندق لكل الفريق
- [ ] طلب غرف متجاورة إن أمكن
- [ ] تأكيد إنترنت سريع في الغرف
- [ ] طلب early check-in إن كان الوصول صباحاً
- [ ] تأكيد شروط الإلغاء

### النقل المحلي
- [ ] ترتيب استقبال من المطار
- [ ] حجز سيارة/حافلة لنقل الفريق
- [ ] تأكيد حجم السيارة (تناسب العدد + الأمتعة)
- [ ] ترتيب نقل يومي (فندق ← الملعب ← فندق)
- [ ] رقم سائق الطوارئ

### الأجهزة والمعدات
- [ ] حجز شحن الأجهزة الثقيلة إن لزم
- [ ] قائمة معدات كل لاعب
- [ ] تأمين أجهزة احتياطية في المقصد
- [ ] محولات كهرباء إن لزم
- [ ] حقيبة إسعافات أولية

### التأمين والوثائق
- [ ] تأمين سفر طبي
- [ ] نسخ من جوازات السفر
- [ ] خطابات دعوة البطولة
- [ ] تأشيرات صادرة
- [ ] بطاقات هوية المنظمة

## قالب جدول السفر
| الوقت | النشاط | الموقع | المسؤول |
|-------|--------|--------|---------|

| dd/mm HH:MM | Check-in | الفندق | منسق السفر |
