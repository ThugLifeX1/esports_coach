# معايير المونتاج

## معايير الجودة العامة
| العنصر | المعيار |
|--------|---------|
| الدقة | 1080p كحد أدنى، 4K عند توفر المصدر |
| الإطارات | 60fps للمحتوى التنافسي، 30fps للوثائقي |
| الصوت | -14 LUFS كحد أقصى، -1 dBFS true peak |
| الترميز | H.264 للنشر، ProRes للأرشيف |
| الألوان | Rec.709 + تطابق هوية العلامة |

## أنواع الفيديو والمواصفات

### ملخص المباراة (3-5 دقائق)
- افتتاحية: شعار + اسم البطولة (5 ثوان)
- أفضل اللحظات مرتبة زمنياً
- تعليق صوتي أو نصي على الشاشة
- إحصائيات بين اللقطات
- خاتمة: نتيجة + ترتيب + CTA

### Reel/Short (15-60 ثانية)
- Hook في أول 3 ثوان (صوت + صورة)
- لحظة ذروة واحدة محورية
- موسيقى ترندية أو حماسية
- نص على الشاشة واضح وكبير
- CTA في الثواني الأخيرة

### وثائقي قصير (5-10 دقائق)
- هيكل: مقدمة ← صعود ← تحدي ← نتيجة
- مقابلات + لقطات متنوعة + موسيقى درامية
- ترانزيشنات سلسة
- بيئة صوتية نظيفة

### خلف الكواليس (2-4 دقائق)
- لقطات طبيعية وبدون إخراج
- موسيقى خفيفة
- نصوص تعريفية قصيرة
- نهاية ممتعة أو مفاجئة

## قواعد المونتاج
- لا قفزات بدون J/L cut
- الحفاظ على الإيقاع (قطع على الإيقاع الموسيقي)
- Color grading متسق مع هوية العلامة
- Sound design: مؤثرات صوتية للقتالات والأحداث
- Subtitles: عربي + إنجليزي للقنوات الدولية
