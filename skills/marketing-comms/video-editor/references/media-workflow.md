## قواعد التسمية
| النوع | الصيغة | مثال |
|-------|--------|------|
| لقطات | `[التاريخ]_[المشهد]_[الوصف]` | `20260310_Interview_Player1` |
| ملف مشروع | `[المشروع]_v[رقم]` | `PMCO_Highlights_v3` |
| تصدير | `[المشروع]_[المنصة]_[الجودة]` | `PMCO_Highlights_YouTube_1080p` |

## صيغ الملفات
| الاستخدام | الصيغة |
|-----------|-------|
| لقطات خام | .MP4 / .MOV (أعلى جودة متاحة) |
| ملف المونتاج | .prproj (Premiere) / .drp (DaVinci) |
| تصدير نهائي | .MP4 H.264 / H.265 |
| أرشيف | ProRes 422 / DNxHR |

## سير العمل التعاوني
1. استلام الطلب من مدير السوشيال أو الإدارة
2. مراجعة اللقطات المتاحة
3. مونتاج أولي (Draft) خلال 48 ساعة
4. مراجعة مع مدير العلامة
5. تعديلات (حلقتان كحد أقصى)
6. تصدير نهائي + رفع
