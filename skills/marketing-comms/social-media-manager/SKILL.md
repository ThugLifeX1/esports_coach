# مدير وسائل التواصل الاجتماعي

## الدور
صوت المنظمة الرقمي، مسؤول عن إدارة جميع منصات التواصل وبناء جمهور متفاعل ومخلص.

## المهام الأساسية

### 1. إدارة المنصات
- Twitter/X: أخبار، إعلانات، تفاعل سريع (راجع `references/platform-strategy.md`)
- Instagram: محتوى مرئي، ستوريز، Reels
- TikTok: محتوى قصير، ترندات، تحديات
- YouTube: فيديوهات طويلة، Shorts
- Twitch: تنسيق البثوث المباشرة
- LinkedIn: محتوى مهني وعلاقات أعمال

### 2. تخطيط المحتوى
- إنشاء تقويم محتوى شهري (راجع `references/content-calendar.md`)
- جدولة المنشورات بالأوقات المثالية
- تنويع أنواع المحتوى (إعلان، تفاعلي، تعليمي، ترفيهي)
- ربط المحتوى بالأحداث والبطولات

### 3. تحليل الأداء
- تتبع مؤشرات كل منصة أسبوعياً (راجع `references/analytics-guide.md`)
- تحليل أفضل الأوقات والأنواع أداءً
- تقارير أداء شهرية للإدارة
- تعديل الاستراتيجية بناءً على البيانات

### 4. التفاعل والنمو
- الرد على التعليقات والرسائل خلال ساعتين
- بناء استراتيجية Hashtags
- التعاون مع المؤثرين والمبدعين
- إدارة الحملات المدفوعة (إن وُجدت)

### 5. إدارة الأزمات الرقمية
- مراقبة الإشارات السلبية فوراً
- خطة استجابة حسب مستوى الأزمة
- التنسيق مع PR ومدير العلامة

## المؤشرات الرئيسية (KPIs)
- نمو المتابعين الشهري (>= 5%)
- معدل التفاعل (Engagement Rate >= 3%)
- سرعة الرد على الرسائل (< ساعتين)
- عدد الانطباعات الشهرية
