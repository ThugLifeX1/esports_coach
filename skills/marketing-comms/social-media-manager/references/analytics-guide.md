# دليل التحليلات

## المؤشرات الرئيسية لكل منصة

### Twitter/X
| المؤشر | التعريف | المستهدف |
|--------|---------|---------|
| Impressions | عدد مرات الظهور | نمو 10% شهرياً |
| Engagement Rate | تفاعلات / انطباعات | >= 2.5% |
| Profile Visits | زيارات الملف | نمو مستمر |
| Link Clicks | نقرات الروابط | حسب الحملة |

### Instagram
| المؤشر | التعريف | المستهدف |
|--------|---------|---------|
| Reach | عدد الواصلين | نمو 8% شهرياً |
| Engagement Rate | (إعجابات+تعليقات+حفظ) / وصول | >= 3% |
| Story Views | مشاهدات الستوري | >= 60% من المتابعين |
| Reel Plays | مشاهدات Reels | نمو 15% شهرياً |

### TikTok
| المؤشر | التعريف | المستهدف |
|--------|---------|---------|
| Video Views | مشاهدات الفيديو | نمو 20% شهرياً |
| Engagement Rate | تفاعلات / مشاهدات | >= 5% |
| Average Watch Time | متوسط وقت المشاهدة | >= 50% من المدة |
| Follower Growth | نمو المتابعين | >= 8% شهرياً |

### YouTube
| المؤشر | التعريف | المستهدف |
|--------|---------|---------|
| Watch Time | ساعات المشاهدة | نمو 15% شهرياً |
| CTR | نسبة النقر على الصور المصغرة | >= 5% |
| Subscriber Growth | نمو المشتركين | >= 5% شهرياً |
| Average View Duration | متوسط مدة المشاهدة | >= 40% |
