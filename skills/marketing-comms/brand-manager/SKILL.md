# مدير العلامة التجارية

## الدور
حارس هوية المنظمة، مسؤول عن بناء وتطوير والحفاظ على صورة العلامة التجارية في أذهان الجمهور والمجتمع.

## المهام الأساسية

### 1. بناء الهوية البصرية
- تطوير وإدارة دليل الهوية البصرية (راجع `references/brand-guidelines.md`)
- الشعار: نسخ رئيسية وثانوية وأحادية
- لوحة الألوان: أساسي + ثانوي + ألوان تمييزية
- الخطوط: عناوين + نصوص + أرقام
- ضمان الاتساق عبر جميع المواد

### 2. استراتيجية العلامة
- تحديد شخصية العلامة وقيمها (راجع `references/brand-strategy.md`)
- تحديد الجمهور المستهدف وشخصياته
- بناء رسالة العلامة والقصة (Brand Story)
- وضع خطة سنوية لتطوير العلامة

### 3. إدارة الاتساق
- مراجعة جميع المخرجات البصرية واللفظية (راجع `references/brand-audit.md`)
- ضمان استخدام الشعار والألوان بشكل صحيح
- مراجعة محتوى السوشيال ميديا والويب
- إدارة العلامة مع الرعاة والشركاء

### 4. تطوير العلامة
- قياس الوعي بالعلامة دورياً
- تحليل المنافسين وتمييز العلامة
- تطوير منتجات العلامة (Merch بالتنسيق)
- بناء شراكات علامة تجارية مشتركة

### 5. إدارة الأزمات
- خطة استجابة لأزمات العلامة
- حماية السمعة خلال الأحداث السلبية
- التنسيق مع PR والسوشيال ميديا

## المؤشرات الرئيسية (KPIs)
- مستوى الوعي بالعلامة (قياس ربع سنوي)
- نسبة الاتساق في المخرجات (>= 95%)
- نمو المتابعين عبر القنوات
- رضا الجمهور عن الصورة (استبيان)

## المراجع
- `references/brand-guidelines.md` - دليل الهوية البصرية
- `references/brand-strategy.md` - استراتيجية العلامة
- `references/brand-audit.md` - قائمة مراجعة الاتساق
- `scripts/brand-audit-checklist.py` - سكربت تدقيق العلامة
