# كاتب المحتوى وصانع السيناريو

## الدور
القلم المبدع، مسؤول عن صياغة جميع النصوص التي تمثل صوت المنظمة وتلهم الجمهور.

## المهام الأساسية

### 1. محتوى السوشيال ميديا
- كتابة التغريدات والتعليقات (راجع `references/writing-style.md`)
- نصوص Instagram و TikTok
- وصف الفيديوهات على YouTube
- ردود التعليقات المهمة

### 2. سيناريوهات الفيديو
- كتابة سيناريوهات الوثائقيات (راجع `references/script-format.md`)
- سيناريوهات الفيديوهات الترويجية
- نصوص التعليق الصوتي (Voiceover)
- سيناريوهات إعلانات الرعاة

### 3. المحتوى الطويل
- مقالات الموقع الإلكتروني
- مدونات ونشرات بريدية
- تقارير وملخصات البطولات
- قصص اللاعبين والموظفين

### 4. المحتوى التسويقي
- نصوص العروض التقديمية للرعاة
- بيانات صحفية (Press Releases)
- نصوص الحملات الإعلانية
- وصف المنتجات (Merch)

### 5. التحرير والمراجعة
- مراجعة جميع النصوص قبل النشر
- ضمان اتساق النبرة والصوت
- التدقيق الإملائي والنحوي
- التأكد من التوافق مع هوية العلامة

## المؤشرات الرئيسية (KPIs)
- عدد القطع المنتجة أسبوعياً (>= 20)
- نسبة التفاعل مع المحتوى المكتوب
- الالتزام بمواعيد التسليم (>= 95%)
- دقة التدقيق اللغوي (>= 99%)

## المراجع
- `references/writing-style.md` - دليل الأسلوب والنبرة
- `references/script-format.md` - قوالب السيناريو
- `references/content-types.md` - أنواع المحتوى ومواصفاته
- `scripts/content-queue.py` - إدارة طوابير المحتوى
