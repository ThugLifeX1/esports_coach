# دليل الأسلوب والنبرة

## شخصية الكتابة
- **النبرة:** احترافية بحماس، واثقة وليست متعجرفة
- **الصوت:** صوت قائد ملهم - يعرف ما يريد ويشجع الآخرين
- **الأسلوب:** مباشر، حماسي، يثير الفضول

## قواعد الكتابة

### العربية
- استخدام الفصحى المبسطة (ليست معقدة)
- تجنب الترجمات الحرفية من الإنجليزية
- أسماء الألعاب والمصطلحات بالإنجليزية كما هي
- أرقام الهاشتاقات والإحصائيات بالأرقام العربية

### الإنجليزية
- American English
- مصطلحات الـ Esports كما هي متعارف عليها
- لا ترجمة مصطنعة لمصطلحات اللعبة

## نبرة المنصات
| المنصة | النبرة | الطول | أمثلة |
|--------|--------|-------|-------|
| Twitter/X | سريعة، مباشرة | 280 حرف | "الفوز لنا الليلة! 🏆" |
| Instagram | ملهمة، وصفية | 100-300 كلمة | قصص خلف الكواليس |
| TikTok | عفوية، ترندية | نص قصير | "لا تصدق هاللحظة! 😱" |
| YouTube | احترافية، تفصيلية | وصف 200+ كلمة | ملخص شامل |
| LinkedIn | مهنية، تحليلية | 150-400 كلمة | إنجازات وشراكات |
| Press Release | رسمية، موضوعية | 300-600 كلمة | إعلانات مؤسسية |

## كلمات وعبارات المنظمة
- **استخدم:** "نصنع التاريخ" / "نقاتل من أجل..." / "عائلة واحدة" / "القادم أعظم"
- **تجنب:** "نحاول" / "إن شاء الله نفوز" / "مجرد لعبة"
- **CTAs:** "انضموا لمجتمعنا" / "تابعونا" / "شاهدوا اللحظة"

## قوالب جاهزة
### نتيجة فوز
```
🏆 انتصار! فريقنا يحقق [النتيجة] ضد [المنافس]
[إحصائية مميزة]
القادم أعظم! 🔥
#[OrgName]Wins #[Game]Esports
```

### إعلان مباراة
```
⚔️ مواجهة حاسمة!
📅 [التاريخ] | 🕐 [الوقت]
.vs [المنافس]
شاهدونا مباشرة على [المنصة]
#[OrgName] #MatchDay
```

### ترحيب بلاعب
```
🤝 مرحباً [الاسم] في عائلة [المنظمة]!
[جملة عن إنجازاته]
معاً نصنع التاريخ 🚀
#Welcome[الاسم] #[OrgName]
```
