# قوالب السيناريو

## سيناريو فيديو قصير (Reel/Short)

```
المنصة: [TikTok/Instagram/YouTube Shorts]
المدة: [15-60 ثانية]

[0:00-0:03] HOOK
الصورة: [وصف اللقطة]
الصوت: [مؤثر/موسيقى]
النص: [النص على الشاشة]

[0:03-0:15] الجسم
الصورة: [وصف]
الصوت: [تعليق/موسيقى]
النص: [النص]

[0:15-0:20] الخاتمة + CTA
الصورة: [لقطة ختامية]
الصوت: [مؤثر ختامي]
النص: [CTA]
```

## سيناريو ملخص مباراة (3-5 دقائق)

```
العنوان: [عنوان الفيديو]
المنصة: YouTube
المدة المستهدفة: [3-5 دقائق]

[0:00-0:05] INTRO
شعار المنظمة + اسم البطولة
موسيقى: حماسية إلكترونية

[0:05-0:30] سياق المباراة
تعليق صوتي: "اليوم نواجه [المنافس] في [البطولة]..."
لقطات: تدريب + تحضير

[0:30-X:XX] اللحظات الرئيسية
لكل لحظة:
  - الوقت في المباراة
  - وصف اللقطة
  - نص على الشاشة (إحصائية/اسم لاعب)
  - مؤثر صوتي للقتال

[X:XX-X:XX] النتيجة والخاتمة
ترتيب الفريق + إحصائيات
CTA: "تابعونا في المباراة القادمة"
```

## سيناريو وثائقي (5-10 دقائق)

```
العنوان: [عنوان الوثائقي]
المنصة: YouTube
المدة المستهدفة: [5-10 دقائق]

الفصل 1: المقدمة (1-2 دقيقة)
- Hook: لحظة درامية من القصة
- سياق: من نحن وما القصة

الفصل 2: البداية (1-2 دقيقة)
- الخلفية والبدايات
- مقابلة/تعليق

الفصل 3: التحدي (2-3 دقائق)
- العقبة الرئيسية
- لقطات مكثفة + موسيقى درامية

الفصل 4: النتيجة (1-2 دقيقة)
- الحل والانتصار (أو الدرس)
- لقطات احتفالية/تأملية

الفصل 5: الخاتمة (30-60 ثانية)
- رسالة ختامية
- CTA
```

## سيناريو إعلان راعي (15-60 ثانية)
- يجب أن يبدأ كمحتوى عادي ثم يدمج الراعي بشكل طبيعي
- ذكر اسم الراعي + ما يقدمه + CTA
- لا يزيد عن 15 ثانية من المحتوى التجاري
