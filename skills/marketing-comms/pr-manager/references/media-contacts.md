# قاعدة بيانات الإعلاميين

## تصنيف الإعلاميين
| الفئة | الوصف | الأولوية |
|--------|------|---------|
| رئيسي | صحفيون في منصات esports كبرى | عالية |
| ثانوي | صحفيون في منصات رياضية/تقنية عامة | متوسطة |
| محلي | صحفيون في وسائل إعلام محلية | متوسطة |
| مؤثرون | صناع محتوى ذات تأثير إعلامي | عالية |
| مستقلون | كتاب ومحللون مستقلون | منخفضة |

## قالب التوثيق
| الحقل | الوصف |
|-------|-------|
| الاسم | الاسم الكامل |
| المنصة | الجهة التي يعمل بها |
| التخصص | Esports / تقنية / رياضة عامة |
| المنطقة | جغرافية التغطية |
| البريد | للتواصل الرسمي |
| السوشيال | حسابات التواصل |
| آخر_تواصل | تاريخ آخر تفاعل |
| العلاقة | □ ممتازة □ جيدة □ جديدة □ تحت بناء |

## قنوات التواصل
- بريد إلكتروني: للبيانات الصحفية والإعلانات الرسمية
- Twitter/X DM: للتواصل السريع والمتابعة
- WhatsApp/Telegram: للعلاقات الشخصية المتبادلة
- LinkedIn: للعلاقات المهنية الجديدة

## بروتوكول التواصل
- التحديث الدوري: مرة كل شهر على الأقل
- إرسال أخبار حصرية للعلاقات المميزة
- دعوة للفعاليات والمؤتمرات
- شكر على التغطيات الإيجابية

## قائمة المنصات المستهدفة
- Dot Esports, HLTV, Liquipedia
- IGN Arabia, IGN Middle East
- العربية.نت، الشرق الرياضية
- منصات Esports محلية
- قنوات يوتيوب Esports متخصصة
