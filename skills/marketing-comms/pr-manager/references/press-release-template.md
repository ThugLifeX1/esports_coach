# قالب البيان الصحفي

## الهيكل

```
[للنشر الفوري / تحت حظر حتى التاريخ]

بيان صحفي: [العنوان الجذاب]

[المدينة، الدولة] — [التاريخ] — [الفقرة الأولى: من، ماذا، متى، أين، لماذا في جملتين]

[الفقرة الثانية: تفاصيل إضافية وأرقام مهمة]

[الفقرة الثالثة: اقتباس من مسؤول أو شخص معني]
"[اقتباس مباشر من المدير التنفيذي/المدرب/اللاعب]" — [الاسم، المنصب]

[الفقرة الرابعة: معلومات إضافية أو سياق]

[الفقرة الخامسة: تفاصيل عملية - أين يمكن للمتابعين المشاهدة/الانضمام]

### عن [اسم المنظمة]
[2-3 جمل تعريفية بالمنظمة]

وسائل التواصل:
- الموقع: [URL]
- Twitter/X: [@handle]
- Instagram: [@handle]
- بريد الإعلام: press@[org].com

جهة الاتصال الإعلامي:
[الاسم] — مدير العلاقات العامة
[البريد] | [الهاتف]
```

## أنواع البيانات
| النوع | متى | مثال عنوان |
|-------|------|-----------|
| توقيع لاعب | عند التعاقد | "[المنظمة] تعلن انضمام [اللاعب] لفريق [اللعبة]" |
| شراكة/رعاية | عند التوقيع | "[المنظمة] و[الشركة] يعلنان شراكة استراتيجية" |
| نتيجة بطولة | بعد الفوز | "[المنظمة] تحقق [المركز] في [البطولة]" |
| إطلاق مشروع | عند الإطلاق | "[المنظمة] تطلق برنامج الأكاديمية لاكتشاف المواهب" |
| أزمة | عند الحاجة | "بيان رسمي بخصوص [الموضوع]" |

## نصائح
- عنوان واضح ومباشر
- جملة رئيسية تلخص الخبر
- اقتباسات حقيقية وليست مُصطنعة
- لا يتجاوز صفحة واحدة
- إرسال قبل الحدث بـ 24-48 ساعة (under embargo)
