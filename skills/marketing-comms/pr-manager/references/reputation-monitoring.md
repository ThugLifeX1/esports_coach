## تصنيف الإشارات
| النوع | الوصف | الإجراء |
|-------|------|---------|
| إيجابي | مدح أو تغطية جيدة | شكر + مشاركة |
| محايد | ذكر بدون رأي | مراقبة |
| سلبي خفيف | نقد بنّاء أو شكوى | رد بنّاء |
| سلبي شديد | هجوم أو اتهام | رد رسمي + تنسيق |
| أزمة | تهديد أو فضيحة | بروتوكول الأزمة فوراً |

## بروتوكول الاستجابة للأزمات

### المستوى 1: تنبيه (ساعات)
- إشارة سلبية محدودة الانتشار
- الإجراء: رد بنّاء + مراقبة
- المسؤول: PR Manager

### المستوى 2: تحذير (2-4 ساعات)
- إشارة سلبية تنتشر بسرعة
- الإجراء: بيان رسمي + تنسيق مع الإدارة
- المسؤول: PR Manager + CEO

### المستوى 3: أزمة (فوري)
- أزمة كبرى تهدد السمعة
- الإجراء: تجميع فريق الأزمة + بيان عاجل
- المسؤول: CEO + Legal + PR

## خطة فريق الأزمة
| العضو | الدور |
|-------|-------|
| CEO | القرار النهائي |
| PR Manager | التوجه الإعلامي + البيانات |
| Legal Advisor | المراجعة القانونية |
| Social Media Manager | التنفيذ على المنصات |
| مدير العلامة | ضمان الاتساق |
