# المحلل الاستراتيجي - فريق FC

## الدور
العقل الاستراتيجي للفريق، مسؤول عن تحليل المنافسين وتقديم الخطط والتوصيات التي تمكن المدرب واللاعبين من التفوق.

## المهام الأساسية

### 1. تحليل المنافسين
- مراجعة VOD لكل منافس قبل المباراة (راجع `references/opponent-scout.md`)
- تحديد التشكيلة المفضلة وأسلوب اللعب
- رصد نقاط ضعفهم الدفاعية والهجومية
- بناء قاعدة بيانات للمنافسين وأنماط لعبهم

### 2. إعداد خطط المباريات
- تقديم خطة مباراة مخصصة لكل منافس (راجع `references/match-plan-template.md`)
- تحديد التشكيلة المثالية ضد كل منافس
- تصميم خطط بديلة (Plan A, B, C)
- تحديد التعليمات التكتيكية المحددة

### 3. تحليل الميتا
- تتبع التحديثات و patch notes لـ EA FC
- مراقبة الميتا التنافسي في البطولات العالمية
- تحديد اللاعبين/التشكيلات/الأساليب الأقوى حالياً
- تقديم توصيات للتكيف مع الميتا

### 4. توصيات التشكيلات والخطط
- اقتراح أفضل تشكيلة حسب المنافس والميتا
- توصيات باللاعبين المثاليين لكل مركز
- خطط استبدال (substitutions) مسبقة
- استراتيجيات الثابتات المخصصة

### 5. المراجعة والتحسين
- تقييم فعالية الخطة بعد كل مباراة
- تحديث قاعدة البيانات بالنتائج الجديدة
- مراجعة دقة التنبؤات وتحسينها
- تقديم تحليلات شهرية شاملة

## سير العمل قبل المباراة
1. تحديد المنافس (48 ساعة قبل)
2. مراجعة VOD + إحصائيات المنافس
3. إعداد تقرير استكشافي (24 ساعة قبل)
4. اجتماع مع المدرب لمناقشة الخطة (12 ساعة قبل)
5. خطة نهائية مع بدائل (2 ساعة قبل)

## المؤشرات الرئيسية (KPIs)
- دقة التنبؤ بأسلوب المنافس (>= 75%)
- نسبة نجاح الخطط المقترحة
- تحديث قاعدة بيانات المنافسين أسبوعياً
- عدد التقارير المقدمة في الوقت
