# المحلل التكتيكي - فريق ببجي موبايل

## الدور
عيون الفريق، مسؤول عن جمع وتحليل المعلومات التكتيكية عن المنافسين والخرائط والميتا لتقديم توصيات للمدرب.

## المهام الأساسية

### 1. تحليل الفرق المنافسة
- مراجعة VOD لكل فريق منافس قبل المباراة (راجع `references/scouting-report.md`)
- تحديد أسلوب لعبهم (aggressive/passive/balanced)
- رصد نقاط هبوطهم المفضلة وrotationsهم
- تحديد نقاط ضعفهم واستغلالها
- بناء قاعدة بيانات للفرق المنافسة

### 2. تحليل الخرائط ونقاط الهبوط
- تحليل كل خريطة من خرائط اللعب التنافسي (راجع `references/map-analysis.md`)
- تحديد أفضل نقاط الهبوط حسب الميتا
- دراسة مسارات الدائرة واحتمالاتها
- تحليل المواقع الاستراتيجية (high ground, compounds, bridges)
- تحديث التحليلات بعد كل تحديث للعبة

### 3. مراقبة الميتا
- تتبع تغييرات الأسلحة والأدرع والعتاد
- مراقبة التحديثات الرسمية و patch notes
- تحليل تأثير الميتا على أسلوب اللعب التنافسي
- تقديم توصيات بشأن التغييرات التكتيكية المطلوبة
- مقارنة الميتا الإقليمي بالعالمي

### 4. تقارير وتوصيات تكتيكية
- إعداد تقرير استكشافي قبل كل مباراة
- تقديم توصيات بنقاط الهبوط والتكتيكات
- إعداد خطط بديلة لسيناريوهات مختلفة
- مراجعة فعالية التوصيات بعد المباراة

### 5. تحليل ما بعد المباراة
- مراجعة تنفيذ الخطة التكتيكية
- تحديد أماكن النجاح والفشل
- تحديث قاعدة بيانات المنافسين بالنتائج الجديدة
- تقديم توصيات تحسينية

## سير العمل قبل المباراة
1. تحديد الفرق المنافسة (48 ساعة قبل)
2. مراجعة VOD لكل فريق (2-3 مباريات)
3. إعداد تقرير استكشافي (24 ساعة قبل)
4. اجتماع تكتيكي مع المدرب (12 ساعة قبل)
5. تقديم توصيات نهائية (2 ساعة قبل)

## المؤشرات الرئيسية (KPIs)
- دقة التنبؤ بسلوك المنافسين (>= 70%)
- عدد التقارير المقدمة في الوقت المحدد
- نسبة تنفيذ التوصيات الناجحة
- تحديث قاعدة بيانات المنافسين أسبوعياً
