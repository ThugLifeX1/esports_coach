# دليل منع الاحتراق الوظيفي (Burnout)

## مراحل الاحتراق

### المرحلة 1: الإرهاق (Strain)
- شعور بالتعب المزمن
- تراجع طفيف في الحماس
- صعوبة في بدء التدريب
- **التدخل:** تعديل الجدول + تقنيات استرخاء

### المرحلة 2: التبلد (Blunting)
- فقدان المتعة في اللعب
- أداء آلي بدون إبداع
- شعور بالرتابة
- **التدخل:** إجازة قصيرة (2-3 أيام) + تنويع الأنشطة

### المرحلة 3: الاحتراق (Burnout)
- كراهية اللعب
- أعراض جسدية (صداع، أرق)
- انعزال عن الفريق
- **التدخل:** إجازة طويلة (1-2 أسبوع) + علاج متخصص

## مؤشرات الإنذار المبكر

### سلوكية
| المؤشر | طبيعي | تنبيه | خطر |
|--------|--------|-------|------|
| ساعات اللعب اليومية | 6-8 | 10+ أو < 3 | > 12 أو رفض اللعب |
| المزاج أثناء التدريب | متحمس | محايد/ساخط | كئيب/غاضب |
| العلاقة مع الفريق | إيجابية | انسحابية | عدوانية |
| رد الفعل للخسارة | تحفيز | إحباط | يأس |

### جسدية
- اضطرابات النوم
- صداع متكرر
- آلام الرقبة والظهر
- فقدان/زيادة الشهية
- إرهاق مستمر

### أدائية
- تراجع مفاجئ في الإحصائيات
- أخطاء غير معتادة
- بطء في اتخاذ القرارات
- عدم القدرة على التركيز

## خطة الوقاية

### يومية
- [ ] فترات راحة كل 90 دقيقة (10 دقائق)
- [ ] تمارين عين 20-20-20 (كل 20 دقيقة)
- [ ] حركة جسدية (تمدد/مشي) 3 مرات يومياً
- [ ] وقت بدون شاشات قبل النوم بساعة

### أسبوعية
- [ ] يوم راحة واحد على الأقل
- [ ] نشاط خارج اللعب (رياضة، هواية)
- [ ] جلسة مع الأخصائي النفسي
- [ ] تقييم ذاتي لمستوى الإرهاق (1-10)

### شهرية
- [ ] استبيان الرفاهية الشامل
- [ ] مراجعة ساعات اللعب مع المدرب
- [ ] تقييم الحاجة لإجازة
- [ ] تعديل الأهداف إن لزم

## بروتوكول التعافي
1. إيقاف التدريب التنافسي فوراً
2. إجازة كاملة (مدة حسب المرحلة)
3. أنشطة بديلة (رياضة، طبيعة، اجتماعية)
4. عودة تدريجية (50% → 75% → 100%)
5. متابعة مستمرة لمدة 3 أشهر
