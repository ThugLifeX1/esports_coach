# قائمة السفر والبعثات

## قبل السفر بـ 4 أسابيع
- [ ] تأكيد مشاركة الفريق في البطولة
- [ ] حجز التذاكر (طيران)
- [ ] حجز الإقامة (فندق قريب من الموقع)
- [ ] التقدم للتأشيرات (إن لزم)
- [ ] حجز النقل المحلي (مطار → فندق → الموقع)

## قبل السفر بـ 2 أسبوع
- [ ] استلام التأشيرات
- [ ] تأكيد الحجوزات جميعها
- [ ] تجهيز وثائق اللاعبين:
  - [ ] جوازات السفر (صلاحية >= 6 أشهر)
  - [ ] بطاقات الهوية الوطنية
  - [ ] خطابات الدعوة من المنظمين
  - [ ] تأمين سفر
- [ ] إعداد قائمة المعدات (راجع أدناه)

## قبل السفر بـ 3 أيام
- [ ] تأكيد نهائي مع المنظمين
- [ ] مراجعة جدول البطولة كاملاً
- [ ] التأكد من صحة جميع الحجوزات
- [ ] إرسال برنامج الرحلة للفريق
- [ ] مراجعة قواعد البطولة والمنشطات

## يوم السفر
- [ ] تجمع الفريق قبل الموعد بـ 3 ساعات
- [ ] التأكد من حمل جميع الوثائق
- [ ] فحص المعدات الإلكترونية
- [ ] التواصل مع المنظمين عند الوصول

## قائمة المعدات
- [ ] أجهزة اللعب (هواتف/أجهزة تحكم)
- [ ] شواحن وكابلات احتياطية
- [ ] سماعات الرأس (2 لكل لاعب)
- [ ] محولات طاقة (Power adapters)
- [ ] أجهزة التوصيل (Dongles/Hubs)
- [ ] كابلات شبكة (Ethernet)
- [ ] أجهزة تسجيل (إن لزم)
- [ ] ملابس الفريق الرسمية

## عند العودة
- [ ] تسوية المصروفات مع المالية
- [ ] كتابة تقرير الرحلة
- [ ] توثيق الدروس المستفادة
- [ ] تحديث سجل السفر
