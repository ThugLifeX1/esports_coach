# قائمة الالتزام بالقواعد التنافسية

## قواعد اللاعبين
- [ ] العمر المؤهل (>= 16 أو حسب البطولة)
- [ ] حساب واحد فقط لكل لاعب (لا تعدد حسابات)
- [ ] عدم استخدام برامج غش أو تعديل (Hacks/Mods)
- [ ] عدم مشاركة الحساب مع أي شخص آخر
- [ ] الالتزام بقواعد السلوك (لا بذاءة، لا عنصرية)

## قواعد الفريق
- [ ] عدد اللاعبين المطلوب (4 أساسي + 1-2 احتياط)
- [ ] لا يجمع لاعبين من حسابات محظورة سابقاً
- [ ] التسجيل باسم الفريق الرسمي فقط
- [ ] التزام قائد الفريق بالتمثيل الرسمي

## قواعد المعدات
- [ ] أجهزة معتمدة فقط (حسب قائمة البطولة)
- [ ] لا peripherals غير مصرح بها
- [ ] إعدادات اللعبة ضمن الحدود المسموحة
- [ ] فحص الأجهزة قبل كل مباراة

## قواعد الاتصال
- [ ] استخدام قنوات الاتصال المعتمدة فقط
- [ ] لا اتصال خارجي أثناء المباراة
- [ ] التزام بقواعد البث المباشر (إن وُجد)

## قواعد السلوك العام
- [ ] عدم إلقاء المباريات (Match Fixing)
- [ ] عدم المراهنة على النتائج
- [ ] احترام المنظمين والفرق الأخرى
- [ ] الالتزام بالحضور في المواعيد المحددة

## متابعة التحديثات
- [ ] مراجعة تحديثات PUBG Mobile التنافسية أسبوعياً
- [ ] متابعة تغييرات قواعد البطولات قبل كل مشاركة
- [ ] إبلاغ الفريق بأي تغيير خلال 24 ساعة

1. توثيق الحادثة فوراً
2. إبلاغ الإدارة العليا

4. التعاون مع المنظمين في التحقيق
