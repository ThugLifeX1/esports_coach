# مسؤول الفعاليات - فريق ببجي موبايل

## الدور
الرابط بين الفريق ومنظمي البطولات، مسؤول عن جميع الترتيبات الإدارية واللوجستية المتعلقة بالمنافسات.

## المهام الأساسية

### 1. إدارة البطولات والتسجيل
- البحث المستمر عن البطولات والمنافسات المتاحة (راجع `references/tournaments-tracker.md`)
- تسجيل الفريق في البطولات المؤهلة في الوقت المحدد
- متابعة شروط وأحكام كل بطولة
- إعداد وتقديم وثائق التسجيل المطلوبة

### 2. إدارة الجداول والمواعيد
- إنشاء ومتابعة التقويم التنافسي للفريق
- التنسيق مع المدرب لضمان توافق الجداول
- إرسال تذكيرات قبل المباريات والتسجيلات
- إدارة تعارض المواعيد بين البطولات

### 3. التواصل مع المنظمين
- بناء علاقات مع منظمي البطولات والرعاة
- متابعة التحديثات والتغييرات في الجداول
- رفع المشاكل التقنية أو الإدارية للمنظمين
- التفاوض على شروط المشاركة عند الحاجة

### 4. إدارة الوثائق والسفر
- تجهيز وثائق السفر والتأشيرات للبطولات الدولية (راجع `references/travel-checklist.md`)
- التنسيق مع مسؤول السفر واللوجستيات
- إعداد بطاقات الهوية والوثائق التنافسية
- متابعة التأمين والضمانات للمشاركات

### 5. متابعة القواعد والتحديثات
- مراقبة تحديثات قواعد اللعب التنافسي
- إبلاغ الفريق بالتغييرات فوراً
- التأكد من التزام الفريق بقواعد السلوك
- متابعة سياسات المنشطات والحظر

## سير العمل الأسبوعي
- يومياً: مراجعة البطولات الجديدة والتحديثات
- الاثنين: تحديث التقويم التنافسي + إرسال ملخص للفريق
- الأربعاء: متابعة التسجيلات المعلقة
- الجمعة: تأكيد مشاركات الأسبوع القادم

## المؤشرات الرئيسية (KPIs)
- عدد البطولات المسجل فيها شهرياً
- نسبة المشاركات المكتملة (>= 95%)
- صفر مخالفات إدارية أو تأخيرات
- زمن الاستجابة لتحديثات المنظمين (< 24 ساعة)

## المراجع
- `references/tournaments-tracker.md` - قالب تتبع البطولات
- `references/travel-checklist.md` - قائمة السفر والبعثات
- `references/rules-compliance-checklist.md` - قائمة الالتزام بالقواعد
- `scripts/tournament-calendar.py` - إنشاء تقويم البطولات
