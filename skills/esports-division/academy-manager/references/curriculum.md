# المناهج التدريبية للأكاديمية

## المستوى 1: تأسيسي (4-8 أسابيع)

### الأهداف
- إتقان الأساسيات الميكانيكية
- فهم أساسيات اللعب الجماعي
- بناء عادات التدريب المنتظم

### الجدول الأسبوعي (10 ساعات)
| اليوم | المدة | المحتوى |
|-------|-------|---------|
| الاثنين | 2 ساعة | أساسيات + تدريب فردي |
| الثلاثاء | 2 ساعة | تدريب جماعي مبسط |
| الأربعاء | 2 ساعة | مهارات ميكانيكية |
| الخميس | 2 ساعة | سكريم مبتدئ |
| الجمعة | 2 ساعة | مراجعة + تقييم ذاتي |

### مهارات المستوى
- [ ] الأسلحة الأساسية (AR + SMG)
- [ ] الحركة والتنقل
- [ ] Callouts الأساسية
- [ ] Loot الأساسي
- [ ] Map Awareness أولي

## المستوى 2: متوسط (6-12 أسبوع)

### الأهداف
- تحسين الأداء الفردي والجماعي
- فهم التكتيكات المتوسطة
- بناء الوعي التنافسي

### الجدول الأسبوعي (14 ساعة)
| اليوم | المدة | المحتوى |
|-------|-------|---------|
| الاثنين | 2.5 ساعة | ميكانيك متقدم |
| الثلاثاء | 2.5 ساعة | تكتيك + تدريب جماعي |
| الأربعاء | 2.5 ساعة | سكريم + مراجعة |
| الخميس | 2.5 ساعة | سكريم + تحليل |
| الجمعة | 2 ساعة | VOD + تطوير ذاتي |
| السبت | 2 ساعة | منافسة داخلية |

### مهارات المستوى
- [ ] أسلحة متعددة باتقان
- [ ] Rotations واعية
- [ ] قراءة الدائرة
- [ ] التواصل المتقدم
- [ ] الثبات تحت الضغط

## المستوى 3: متقدم (8-16 أسبوع)

### الأهداف
- الاستعداد للترقية للفريق الرئيسي
- إتقان التكتيكات المتقدمة
- بناء اللياقة الذهنية التنافسية

### الجدول الأسبوعي (18 ساعة)
- يتطابق تقريباً مع جدول الفريق الرئيسي
- إضافة جلسات مع مدرب الفريق الرئيسي أسبوعياً
- سكريمات مع فرق خارجية أسبوعياً

### مهارات المستوى
- [ ] إتقان التشكيلات والخطط
- [ ] أداء مستقر تحت الضغط
- [ ] قيادة داخلية (IGL potential)
- [ ] التزام مهني كامل
- [ ] جاهزية للمنافسة الرسمية

## معايير الترقية بين المستويات
| المعيار | من 1→2 | من 2→3 | من 3→الفريق |
|---------|--------|--------|------------|
| K/D | >= 2.0 | >= 3.0 | >= 4.0 |
| الحضور | >= 85% | >= 90% | >= 95% |
| السلوك | مقبول | جيد | ممتاز |
| مدة البرنامج | 4-8 أسابيع | 6-12 أسبوع | 8-16 أسبوع |
