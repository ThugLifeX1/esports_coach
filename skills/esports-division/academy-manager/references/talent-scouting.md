# دليل اكتشاف المواهب

## مصادر الاكتشاف

### مصادر مباشرة
| المصدر | الطريقة | التردد |
|--------|---------|--------|
| Leaderboards اللعبة | مراقبة أفضل 1000 لاعب | أسبوعي |
| بطولات المجتمع | الحضور والمراقبة | حسب الجدول |
| Discord مجتمعنا | قناة تجارب الأداء | يومي |
| Social Media | تتبع هاشتاقات المنظمة | يومي |
| توصيات اللاعبين | برنامج الإحالة | مستمر |

### مصادر غير مباشرة
| المصدر | الطريقة | التردد |
|--------|---------|--------|
| Streaming Platforms | مراقبة البثوث الواعدة | أسبوعي |
| YouTube | بحث عن محتوى تنافسي | أسبوعي |
| بطولات المدارس/الجامعات | التنسيق مع المنظمين | فصلي |

## معايير الفرز الأولي

### المعايير الأساسية
- العمر: 14-22 سنة
- الترتيب: Conqueror / Grandmaster أو ما يعادلها
- K/D Ratio: >= 2.5
- ساعات اللعب: >= 1000 ساعة

### المعايير السلوكية
- سلوك إيجابي في المجتمع
- لا تاريخ حظر أو مخالفات
- التزام بالقواعد التنافسية
- رغبة حقيقية في الاحتراف

## عملية التواصل
1. مراسلة أولية عبر DM
2. تقديم المنظمة وبرنامج الأكاديمية
3. دعوة لجلسة تعريفية (Discord)
4. تقييم مبدئي عبر مباراة ودية
5. دعوة رسمية للأكاديمية (إن اجتاز)

## نموذج رسالة التواصل
```
السلام عليكم [اسم اللاعب]،
لاحظنا أداءك المتميز في [البطولة/الترتيب] وأردنا التواصل معك.
نحن [اسم المنظمة] لدينا برنامج أكاديمية للاعبين الواعدين.
هل تود الانضمام لجلسة تعريفية قصيرة؟
```

## أيام التجربة المفتوحة
- التردد: شهري
- المدة: يوم واحد (4 ساعات)
- عدد المشاركين: 20-30
- البرنامج: تدريب + تقييم + مقابلة
