# طرق التحليل من مصادر متعددة

## التحليل من الفيديو

### فيديو مرفوع مباشرة
| الخطوة | الوصف | الأداة |
|--------|-------|--------|
| 1 | استلام ملف الفيديو | رفع عبر السكربت أو نسخ لمجلد التحليل |
| 2 | استخراج الإطارات المهمة | FFmpeg - Screenshot كل 5 ثوان |
| 3 | التعرف على الإحصائيات المرئية | OCR (Tesseract) على صور HUD اللعبة |
| 4 | تدوين الأحداث الرئيسية | مراجعة يدوية أو AI لاستخراج Kills/Damage |
| 5 | ربط البيانات باللاعب المطابق | مطابقة IGN مع قائمة الفريق |

### رابط فيديو من منصات السوشيال
| المنصة | طريقة الاستخراج |
|--------|----------------|
| YouTube | yt-dlp لتحميل الفيديو ثم التحليل |
| Twitch |yt-dlp أو Twitch CLI |
| TikTok | yt-dlp |

### ما يمكن استخراجه من الفيديو
- Kills والقتالات لكل لاعب
- نقاط الهبوط والدوران (Rotations)
- قرارات IGL (متى Push / Retreat)
- أوقات البقاء (Survival)
- أخطاء وتأخيرات في القرارات

## التحليل من الصور (Screenshots)

### أنواع الصور المدعومة
| نوع الصورة | البيانات القابلة للاستخراج |
|-----------|--------------------------|
| شاشة النتائج (Results) | الترتيب، Kills، Damage، Survival Time |
| لوحة الإحصائيات | K/D، Headshot%، إحصائيات الموسم |
| خريطة اللعبة | نقطة الهبوط، الدوران، Zone |
| شاشة الفريق | أسماء اللاعبين، الحالة (حياة/وفاة) |

### خطوات التحليل
1. استلام الصورة (رفع أو لصق)
2. OCR لاستخراج النصوص والأرقام
3. مطابقة الأسماء مع قائمة الفريق
4. تسجيل البيانات في سجل التحليل
5. تحديث لوحات المتابعة

## التحليل من الجداول والمستندات

### ملفات CSV / Excel
```
الصيغة المتوقعة:
التاريخ,IGN,Kills,Deaths,Damage,Headshots,Placement,Survival_Time
2026-05-17,xAhmed7,8,1,450,3,1,1890
```
- استيراد مباشر عبر السكربت
- تحقق تلقائي من أن IGN موجود في القائمة
- حساب K/D و Headshot% تلقائياً

### Google Sheets
- ربط عبر API أو تصدير CSV
- تحديث دوري (يومي/أسبوعي)

### ملفات PDF / DOCX
- استخراج الجداول من PDF (pdfplumber)
- قراءة نص من DOCX
- مناسب للتقارير البطولات الرسمية

## تحليل شامل - تجميع المصادر

### نموذج دمج البيانات
```
فيديو المباراة → Kills + Rotations + القرارات
صور النتائج → أرقام دقيقة (Damage, Placement)
جدول CSV → بيانات تاريخية منظمة
═══════════════════════════════════════
= تقرير شامل لكل لاعب في المباراة
```

### أولوية المصادر عند التعارض
1. إحصائيات اللعبة الرسمية (الأدق)
2. جدول رقمي (CSV/Sheets)
3. صورة Screenshot
4. فيديو (أقل دقة للأرقام لكن الأفضل للقرارات)

## تكرار التحليل

| النوع | التكرار | الغرض |
|-------|---------|-------|
| بعد كل سكريم | فوري | ملاحظات سريعة للمدرب |
| أسبوعي | كل أحد | لوحة المتابعة الأسبوعية |
| شهري | أول كل شهر | تقرير تطور شامل |
| قبل البطولة | حسب الجدول | تحليل استعدادي |
| بعد البطولة | فوري | تقرير الخلاصة |
