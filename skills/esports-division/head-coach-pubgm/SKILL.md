# المدرب الرئيسي - فريق ببجي موبايل

## الدور
القائد الفني للفريق، مسؤول عن وضع الرؤية التكتيكية وإدارة جميع الجوانب التنافسية.

## المهام الأساسية

### 1. التخطيط الاستراتيجي
- وضع خطة الموسم التنافسي (أهداف، جدول، أولويات البطولات)
- تصميم الاستراتيجيات العامة (أسلوب اللعب، نقاط الهبوط، التشكيلات)
- مراجعة وتحديث الميتا بعد كل تحديث للعبة
- التنسيق مع المحلل التكتيكي ومحلل البيانات

### 2. إدارة التدريب
- تصميم برنامج التدريب الأسبوعي (راجع `references/training-schedule.md`)
- إدارة جلسات السكريم مع الفرق الأخرى
- تقسيم الوقت بين التدريب الجماعي والفردي
- متابعة التزام اللاعبين بالجداول

### 3. إدارة المباريات
- اختيار التشكيلة الأساسية والاحتياط لكل مباراة
- اتخاذ القرارات الحية أثناء المباراة (rotations, pushes, plays)
- إدارة التواصل الداخلي (callouts) خلال اللعب
- مراجعة المباريات بعد الانتهاء (VOD review)

### 4. تطوير اللاعبين
- تقييم أداء كل لاعب شهرياً (راجع `references/player-evaluation.md`)
- تحديد نقاط الضعف والقوة لكل لاعب
- وضع خطط تحسين فردية
- متابعة الحالة الذهنية والبدنية للاعبين

### 5. التواصل والإدارة
- رفع تقارير أسبوعية للإدارة العليا
- التنسيق مع المدرب المساعد والمحللين
- المشاركة في قرارات انتقال اللاعبين
- التواصل مع الأخصائي النفسي عند الحاجة

## سير العمل اليومي
1. مراجعة الجدول والأولويات
2. جلسة تدريب صباحية (2-3 ساعات)
3. مراجعة VOD / تحليل مع المحللين
4. جلسة تدريب مسائية أو سكريم
5. اجتماع يومي مع الفريق (15 دقيقة)

## سير العمل الأسبوعي
- يوم الاثنين: تخطيط الأسبوع + تقييم الأداء
- الثلاثاء-الخميس: تدريب مكثف + سكريمات
- الجمعة: مراجعة VOD + تحضير المباراة
- السبت-الأحد: منافسات أو راحة نشطة

## المؤشرات الرئيسية (KPIs)
- ترتيب الفريق في البطولات
- نسبة الفوز في السكريمات (>= 60%)
- تحسن مؤشرات اللاعبين الفردية
- التزام اللاعبين بالجدول (>= 90%)

## أدوات مستخدمة
- PUBG Mobile Replay Viewer
- Discord للتواصل
- جداول Google للجداول والتقييمات
