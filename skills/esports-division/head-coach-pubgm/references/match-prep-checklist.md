# قائمة تحضير المباراة

## قبل المباراة بـ 48 ساعة
- [ ] مراجعة تحليل الفرق المنافسة من المحلل التكتيكي
- [ ] تحديد الاستراتيجية الرئيسية والبديلة
- [ ] اختيار نقاط الهبوط لكل خريطة
- [ ] مراجعة بيانات أداء الفريق الأخيرة مع محلل البيانات

## قبل المباراة بـ 24 ساعة
- [ ] اجتماع تكتيكي مع الفريق (30 دقيقة)
- [ ] مراجعة VOD للفرق المنافسة
- [ ] التأكد من جاهزية جميع اللاعبين (صحياً ونفسياً)
- [ ] مراجعة قواعد البطولة وتنسيقها

## قبل المباراة بـ 2 ساعة
- [ ] وصول جميع اللاعبين للمنصة/الجهاز
- [ ] فحص الأجهزة والاتصال والصوت
- [ ] إحماء (Warm-up) - مباراة واحدة على الأقل
- [ ] مراجعة سريعة للخطة مع الفريق (10 دقائق)
- [ ] التأكد من تفعيل وضع عدم الإزعاج

## قبل المباراة بـ 15 دقيقة
- [ ] دخول اللوبي
- [ ] التأكد من إعدادات اللعب
- [ ] تذكير الفريق بالنقاط الرئيسية
- [ ] التأكد من جاهزية Discord/الاتصال

## أثناء المباراة
- [ ] التواصل المستمر مع اللاعبين (حسب القواعد)
- [ ] تتبع الدائرة وال rotations
- [ ] تسجيل ملاحظات لكل جولة

## بعد المباراة مباشرة
- [ ] اجتماع سريع (5 دقائق) - انطباعات أولية
- [ ] حفظ الـ Replays
- [ ] إرسال ملاحظات للمحللين

## بعد المباراة بـ 24 ساعة
- [ ] جلسة VOD Review كاملة
- [ ] تحديد الأخطاء والنقاط الإيجابية
- [ ] تحديث الخطة التكتيكية حسب الحاجة
- [ ] كتابة تقرير ما بعد المباراة
