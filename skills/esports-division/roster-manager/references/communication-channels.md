# قنوات التواصل مع اللاعبين

## القنوات الثلاث

### واتساب — العاجل واليومي
| العنصر | التفاصيل |
|--------|---------|
| الغرض | إشعارات سريعة، تذكيرات، تنبيهات، تواصل يومي |
| مجموعة الفريق | مجموعة واتساب تجمع كل اللاعبين + مدير القائمة |
| رسائل فردية | للاعبين الذين لا يستجيبون في المجموعة |
| زمن الاستجابة المتوقع | <= 2 ساعة |
| مثال | "تذكير: اجتماع السكريم الساعة 10 مساءً" |

### تليجرام — الملفات والإعلانات
| العنصر | التفاصيل |
|--------|---------|
| الغرض | مشاركة ملفات، إعلانات رسمية، مستندات، صور |
| قناة الفريق | قناة تليجرام (مدير القائمة كـ Admin) |
| مجموعة النقاش | مجموعة تليجرام للنقاشات الطويلة |
| ميزة | يسمح بملفات كبيرة + تصنيف بالهاشتاقات |
| مثال | إرسال ملف PDF بقواعد البطولة + طلب توقيع |

### البريد الإلكتروني — الرسمي والمستندات
| العنصر | التفاصيل |
|--------|---------|
| الغرض | تعليمات رسمية، مستندات للتوقيع، تقارير، طلبات إدارية |
| القائمة البريدية | قائمة بريدية باسم الفريق |
| زمن الاستجابة المتوقع | <= 12 ساعة |
| مثال | إرسال نموذج تسجيل بطولة + طلب إعادة التوقيع |

## متى تستخدم كل قناة

| الموقف | القناة | السبب |
|--------|--------|-------|
| إشعار جلسة تدريب | واتساب | سريع + يُرى فوراً |
| تعليمات المدرب الرئيسي | واتساب + بريد | عاجل + توثيق رسمي |
| إرسال مستند للتوقيع | تليجرام + بريد | ملف + نسخة رسمية |
| إعلان تسجيل بطولة | تليجرام + واتساب | تفاصيل + إشعار سريع |
| طلب صورة جواز سفر | واتساب أولاً | عاجل غالباً |
| إرسال قواعد البطولة | تليجرام | ملف كبير |
| تنبيه طارئ (إلغاء مباراة) | واتساب + اتصال | أسرع طريق |
| تقرير أسبوعي للاعب | بريد إلكتروني | رسمي + مرجع |

```

الموضوع: [الموضوع]
التفاصيل: [التعليمات]
الموعد النهائي: [التاريخ والوقت]

⚠️ يرجى تأكيد الاستلام بالرد ✓
```

```

📎 أرسل المستند عبر: [واتساب / تليجرام / بريد]
```
