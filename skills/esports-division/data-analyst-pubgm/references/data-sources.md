# مصادر البيانات وكيفية الاستخراج

## المصادر الأساسية

### 1. PUBG Mobile Replay
- طريقة: تسجيل Replay لكل مباراة
- البيانات المتاحة: حركات اللاعبين، القتالات، اللوت
- كيفية الاستخراج: مراجعة يدوية + تسجيل في جدول

### 2. إحصائيات اللعبة الداخلية
- K/D Ratio
- Average Damage per Match
- Headshot Percentage
- Survival Time
- Average Placement
- طريقة: لقطة شاشة بعد كل جلسة + تسجيل يدوي

### 3. منصات الطرف الثالث
- PUBG Lookup (pubglookup.com)
- PUBG OP (pubg.op.gg)
- البيانات المتاحة: إحصائيات مفصلة، تاريخ المباريات

### 4. بيانات السكريمات
- تسجيل يدوي من قبل المدرب المساعد
- البيانات: نتيجة كل جولة، kills، damage، placement

### 5. بيانات البطولات الرسمية
- موقع البطولة الرسمي
- إحصائيات البث المباشر
- تقارير المنظمين

## صيغة التخزين
- ملف JSON لكل مباراة (راجع سكربت performance-tracker.py)
- ملف CSV شهري للتجميع
- قاعدة بيانات رئيسية للتحليل التاريخي
