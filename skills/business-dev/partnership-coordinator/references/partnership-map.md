# خريطة الشراكات المحتملة

## تصنيف الشراكات
| الفئة | الوصف | القيمة | أمثلة |
|--------|------|--------|-------|
| منصات Esports | منظمات ومنصات تنافسية | وصول + مصداقية | ESL, Faceit, FACEIT |
| إعلامية | منصات إعلام وتغطية | وصول + محتوى | قنوات يوتيوب، مواقع أخبار |
| تعليمية | جامعات ومعاهد | تطوير + مجتمع | جامعات محلية، مدارس تقنية |
| حكومية | جهات رياضية حكومية | شرعية + دعم | لجان أولمبية، وزارات رياضة |
| تقنية | شركات تقنية وأجهزة | معدات + خبرة | شركات أجهزة محلية |
| مجتمعية | منظمات مجتمع مدني | أثر + مجتمع | جمعيات شبابية |
