# مسؤول البيع والتجزئة

## الدور
صانع المنتجات الملموسة، مسؤول عن تحويل هوية المنظمة إلى منتجات يرتديها ويستخدمها المجتمع بفخر.

## المهام الأساسية

### 1. تصميم المنتجات
- تطوير خطوط منتجات جديدة (راجع `references/product-catalog.md`)
- تيشيرتات، هوديز، قبعات، سترات
- إكسسوارات (أكواب، سترابات، ملصقات)
- معدات لعب (ماوس باد، ستريم ديك)
- التعاون مع مصمم الجرافيك

### 2. الإنتاج والتصنيع
- اختيار الموردين والتفاوض معهم (راجع `references/production-guide.md`)
- مراقبة جودة الإنتاج
- إدارة مواعيد الإنتاج
- فحص العينات قبل الإنتاج الكمي

### 3. إدارة المتجر
- تشغيل المتجر الإلكتروني (راجع `references/store-operations.md`)
- تحديث المنتجات والمخزون
- إدارة الطلبات والشحن
- خدمة العملاء والردود

### 4. التسويق والمبيعات
- حملات إطلاق المنتجات الجديدة
- خصومات وعروض موسمية
- التعاون مع السوشيال ميديا للترويج
- تحليل المبيعات والطلب

### 5. المخزون والمالية
- تتبع المخزون والطلبات (راجع `references/inventory-guide.md`)
- تنسيق مع المسؤول المالي
- مراقبة التكاليف والهوامش
- تقارير مبيعات شهرية

## المؤشرات الرئيسية (KPIs)
- إيرادات المبيعات الشهرية
- هامش الربح (>= 40%)
- نسبة الطلبات المشحونة في الوقت (>= 95%)
- رضا العملاء (>= 85%)

## المراجع
- `references/product-catalog.md` - كتالوج المنتجات
- `references/production-guide.md` - دليل الإنتاج
- `references/inventory-guide.md` - دليل المخزون
- `scripts/inventory-tracker.py` - تتبع المخزون
